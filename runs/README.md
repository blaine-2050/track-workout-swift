# Self-Test Run Reports

AI-driven self-test runs and gym-test notes land here.

## Format

One markdown file per run: `YYYY-MM-DD-<script-slug>.md`

See [`COMPUTER_USE_PROTOCOL.md`](https://github.com/blaine-2050/track-workout-core/blob/main/COMPUTER_USE_PROTOCOL.md) in the core repo for the report template.

## Screenshots

`runs/screens/` — PNGs referenced by reports. Committed (not gitignored) so they render on GitHub mobile for review.
